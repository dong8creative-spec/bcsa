# 이미지 등록 가이드

## 📋 개요

부산청년사업가들 웹사이트에서 사용되는 모든 이미지는 **ImgBB**를 통해 호스팅됩니다. 각 이미지 타입별 최적 해상도와 업로드 방법을 안내합니다.

---

## 🚀 빠른 시작

### 1. ImgBB API 키 설정

1. **ImgBB 계정 생성**
   - https://imgbb.com 접속
   - 회원가입 및 로그인

2. **API 키 발급**
   - https://api.imgbb.com 접속
   - "Get API Key" 클릭
   - API 키 복사

3. **index.html에 API 키 설정**
   ```javascript
   // index.html 파일 내에서 찾기
   const IMGBB_API_KEY = '여기에_API_키_입력';
   ```

## 📸 이미지 타입별 가이드

### 1. 프로그램 이미지 (총 3장)

**위치**: 관리자 모달 > 프로그램 관리 > 프로그램 등록/수정

**이미지 구성**:
- 대표 이미지: 1장 (필수)
- 추가 이미지: 최대 3장 (선택)

**최적 해상도**:
- **1200x800px** (16:9 비율 권장)
- 파일 크기: 5MB 이하

**업로드 방법**:
1. 관리자 모달 접속
2. "프로그램 관리" 탭 선택
3. "프로그램 등록" 또는 기존 프로그램 "수정" 클릭
4. "대표 이미지 URL" 또는 "프로그램 추가 이미지" 섹션에서 업로드
5. 이미지 선택 후 자동으로 ImgBB에 업로드됨

---

### 2. 프로필 이미지 (1장)

**위치**: 관리자 모달 > 회원 관리 > 회원 등록/수정

**최적 해상도**:
- **400x400px** (정사각형 권장)
- 파일 크기: 5MB 이하

**업로드 방법**:
1. 관리자 모달 접속
2. "회원 관리" 탭 선택
3. "회원 등록" 또는 기존 회원 "수정" 클릭
4. "프로필 이미지" 섹션에서 업로드
5. 이미지 선택 후 자동으로 ImgBB에 업로드됨

---

### 3. 프로그램 후기 이미지 (최대 2장)

**위치**: 커뮤니티 > 프로그램 후기 글 작성

**최적 해상도**:
- 각 **1200x800px** (16:9 비율 권장)
- 파일 크기: 각 5MB 이하

**업로드 방법**:
1. 커뮤니티 메뉴 접속
2. "글 작성" 클릭
3. 카테고리에서 "프로그램 후기" 선택
4. "프로그램 후기 이미지" 섹션에서 최대 2장 업로드
5. 이미지 선택 후 자동으로 ImgBB에 업로드됨

## 💡 이미지 최적화 팁

### 해상도 권장사항

| 이미지 타입 | 권장 해상도 | 비율 | 파일 크기 |
|------------|------------|------|----------|
| 프로그램 이미지 | 1200x800px | 16:9 | 5MB 이하 |
| 프로필 이미지 | 400x400px | 1:1 | 5MB 이하 |
| 프로그램 후기 이미지 | 1200x800px | 16:9 | 5MB 이하 |
| 히어로 배경 | 1920x1080px | 16:9 | 5MB 이하 |
| 통계 배경 | 1920x600px | 16:5 | 5MB 이하 |
| CTA 배경 | 1920x800px | 16:7 | 5MB 이하 |

### 이미지 포맷 권장사항

- **JPEG**: 사진, 복잡한 이미지에 적합
- **PNG**: 투명 배경이 필요한 경우
- **WebP**: 최신 브라우저에서 더 작은 파일 크기 (선택사항)

### 이미지 압축 팁

1. **온라인 도구 사용**
   - TinyPNG: https://tinypng.com
   - Squoosh: https://squoosh.app

2. **포토샵/기타 편집 도구**
   - "웹용으로 저장" 기능 사용
   - 품질 80-90% 권장

## 🔧 문제 해결

### 이미지가 업로드되지 않는 경우

1. **API 키 확인**
   - `index.html`에서 `IMGBB_API_KEY` 값이 올바른지 확인
   - ImgBB 계정에서 API 키가 활성화되어 있는지 확인

2. **파일 크기 확인**
   - 각 이미지는 5MB 이하여야 함
   - 더 큰 파일은 압축 후 업로드

3. **파일 형식 확인**
   - JPG, PNG, GIF, WebP 형식만 지원
   - 다른 형식은 변환 후 업로드

4. **브라우저 콘솔 확인**
   - F12 키를 눌러 개발자 도구 열기
   - Console 탭에서 에러 메시지 확인

### 이미지가 표시되지 않는 경우

1. **URL 확인**
   - 업로드된 이미지 URL이 올바른지 확인
   - ImgBB에서 직접 이미지 URL 접근 가능한지 확인

2. **캐시 문제**
   - 브라우저 캐시 삭제 (Ctrl+Shift+Delete)
   - 하드 리프레시 (Ctrl+F5)

3. **네트워크 문제**
   - 인터넷 연결 확인
   - ImgBB 서버 상태 확인

## 📝 참고사항

### 이미지 URL 직접 입력

모든 이미지 업로드 섹션에서는:
- 파일 업로드 또는
- 이미지 URL 직접 입력

두 가지 방법 모두 지원됩니다.

### 이미지 삭제

- 업로드된 이미지는 각 섹션에서 "X" 버튼을 클릭하여 삭제 가능
- ImgBB에서 직접 삭제하려면 삭제 URL 사용 (개발자 도구에서 확인 가능)

### 이미지 관리

- 모든 이미지는 ImgBB 계정에서 관리 가능
- https://imgbb.com 접속 후 "My Images"에서 확인
