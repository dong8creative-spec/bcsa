# 🎯 빠른 시작 - 로컬 테스트

## 한 줄 명령어로 시작하기

```bash
./start-test.sh
```

또는

```bash
npm run dev
```

---

## 📱 브라우저 접속

서버가 시작되면 브라우저에서 다음 주소로 접속하세요:

```
http://127.0.0.1:5173
```

(포트 번호는 서버 시작 시 터미널에 표시됩니다)

---

## ✅ 즉시 테스트 가능한 기능들

### 1. 회원가입 및 로그인
1. 상단 메뉴에서 **[회원가입]** 클릭
2. 테스트 계정 만들기:
   - 이메일: test@example.com
   - 비밀번호: test1234
   - 이름: 테스트 사용자
3. **[로그인]** 으로 로그인

### 2. 공지사항 보기
1. 메뉴에서 **[공지사항]** 클릭
2. 게시글 목록 확인
3. 게시글 클릭하여 상세 보기

### 3. 커뮤니티
1. 메뉴에서 **[커뮤니티]** 클릭
2. 게시글 목록 확인
3. (로그인 후) 글쓰기 가능

### 4. 맛집 정보
1. 메뉴에서 **[맛집]** 클릭
2. 맛집 목록 확인
3. 카카오맵으로 위치 확인

### 5. 세미나/행사
1. 메뉴에서 **[세미나]** 클릭
2. 일정 확인
3. 캘린더 뷰 확인

### 6. 관리자 페이지
```
http://127.0.0.1:5173/admin
```
(관리자 권한이 필요합니다)

## 🔧 문제 해결

### 포트 충돌 시
```bash
# 다른 포트 사용
npx vite --port 8080
```

### 서버 종료
터미널에서 `Ctrl + C`

### 캐시 문제 시
브라우저에서 `Ctrl + Shift + R` (강제 새로고침)

## 💡 개발자 도구

브라우저에서 `F12` (또는 Mac: `Cmd+Option+I`)를 눌러 개발자 도구를 열고:

1. **Console**: 에러 및 로그 확인
2. **Network**: API 요청/응답 확인
3. **Application**: 로컬 스토리지 확인
4. **Elements**: HTML/CSS 확인
