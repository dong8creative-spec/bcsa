# 검색엔진 노출 가이드 (Google, Naver, Bing 등)

홈페이지(bcsa.co.kr)를 구글·네이버·다음 등 검색엔진에 노출시키는 방법입니다.

---

## 1. 이미 되어 있는 것 (현재 상태)

- **메타 태그**: `index.html`에 `title`, `description`, `keywords`, `canonical` 설정됨
- **Open Graph / Twitter Card**: SNS 공유 시 제목·설명·이미지 표시
- **JSON-LD**: Organization 구조화 데이터 (검색엔진이 사이트 이해에 활용)
- **robots.txt**: `public/robots.txt` — 모든 검색엔진 크롤링 허용 (`Allow: /`)
- **sitemap.xml**: `public/sitemap.xml` — 메인(`/`), 회원가입(`/signup`) 등록
- **네이버**: 사이트 소유확인 메타 태그 적용됨 (`naver-site-verification`)

---

## 2. Google에 노출시키기

### 2-1. Google Search Console 등록

1. **접속**: [Google Search Console](https://search.google.com/search-console)
2. **속성 추가**: “URL 접두어”로 `https://bcsa.co.kr` 입력
3. **소유권 확인** (아래 중 하나 선택)
   - **HTML 태그**: Search Console이 주는 메타 태그를 복사한 뒤, `index.html`의 `<head>` 안에 넣기  
     예: `<meta name="google-site-verification" content="여기_문자열" />`
   - **HTML 파일**: Search Console이 주는 파일명(예: `google123.html`)을 다운로드해서 **배포된 사이트 루트**에 업로드 (예: `public/google123.html` → 배포 후 `https://bcsa.co.kr/google123.html` 접근 가능해야 함)
4. 확인이 끝나면 **“사이트맵” 메뉴**에서 사이트맵 제출  
   - 사이트맵 URL: `https://bcsa.co.kr/sitemap.xml`

### 2-2. 인덱싱 요청 (선택)

- Search Console 상단 “URL 검사”에 `https://bcsa.co.kr` 입력 후 “색인 생성 요청”  
- 새로 만든 중요 페이지도 같은 방식으로 요청 가능

---

## 3. Naver에 노출시키기

- **사이트 소유확인**: 이미 `index.html`에 `naver-site-verification` 메타 태그가 있음
- **검색 등록**: [네이버 서치어드바이저](https://searchadvisor.naver.com) 로그인 → “사이트 등록”에서 `https://bcsa.co.kr` 등록
- **사이트맵 제출**: 서치어드바이저에서 사이트맵 URL `https://bcsa.co.kr/sitemap.xml` 제출 (메뉴명은 “사이트맵 제출” 또는 “요청” 등)

---

## 4. Bing / Daum 등 다른 검색엔진

### Bing (마이크로소프트)

1. [Bing Webmaster Tools](https://www.bing.com/webmasters)
2. 사이트 추가 후 소유권 확인 (HTML 태그 또는 XML 파일 방식 지원)
3. “Sitemaps”에서 `https://bcsa.co.kr/sitemap.xml` 제출

### Daum(다음)

- [다음 검색등록](https://register.search.daum.net/index.daum) 에서 URL 등록
- 사이트맵 제출 가능하면 `https://bcsa.co.kr/sitemap.xml` 제출

---

## 5. 사이트맵(sitemap.xml) 보강 (선택)

검색에 노출시키고 싶은 **로그인 없이 접근 가능한** 페이지가 더 있으면, `public/sitemap.xml`에 `<url>` 항목을 추가하는 것이 좋습니다.

- 예: 소개 페이지, 정모/세미나 목록, 협력기관, 공지 등  
- **비공개/로그인 전용 페이지**는 사이트맵에 넣지 않는 것이 좋습니다.

형식 예:

```xml
<url>
  <loc>https://bcsa.co.kr/경로</loc>
  <changefreq>weekly</changefreq>
  <priority>0.8</priority>
</url>
```

추가 후 **배포**하고, Google Search Console / Naver 서치어드바이저 등에서 “사이트맵 제출”을 다시 하거나 갱신하면 됩니다.

---

## 6. 요약 체크리스트

| 항목 | 상태 | 비고 |
|------|------|------|
| title / description / canonical | ✅ | `index.html` |
| robots.txt | ✅ | `public/robots.txt` |
| sitemap.xml | ✅ | `public/sitemap.xml` |
| Open Graph / Twitter | ✅ | `index.html` |
| JSON-LD (Organization) | ✅ | `index.html` |
| **Google Search Console** | ⬜ 직접 진행 | 속성 추가 → 소유권 확인 → 사이트맵 제출 |
| **Naver 서치어드바이저** | ⬜ 확인 | 소유확인 완료 후 사이트맵 제출 |
| **Bing Webmaster** | ⬜ 선택 | 사이트 추가 후 사이트맵 제출 |

검색 결과에 반영되기까지는 **며칠~몇 주** 걸릴 수 있습니다. Google Search Console과 네이버 서치어드바이저에서 “색인/수집” 상태를 확인하면 진행 상황을 파악할 수 있습니다.
