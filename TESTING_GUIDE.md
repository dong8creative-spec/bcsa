# 테스트 가이드

## 1. 나라장터 API 기능 테스트

### 배포 방법
```bash
# Firebase Functions 배포
cd functions
npm install
cd ..
firebase deploy --only functions

# 프론트엔드 배포
npm run build
firebase deploy --only hosting
```

### 테스트 시나리오

#### 1.1 기본 검색 테스트
1. 사이트 접속 후 "입찰공고" 메뉴 클릭
2. 다음 검색어로 각각 테스트:
   - 한글: "부산", "IT", "시스템"
   - 영문: "system", "software"
   - 특수문자 포함: "SW개발", "IT·정보통신"
3. 예상 결과:
   - 검색 결과가 표시됨
   - 페이지네이션이 정상 작동
   - 로딩 상태 표시

#### 1.2 필터 조합 테스트
1. 업무구분 필터:
   - "물품", "일반용역", "공사" 각각 선택하여 검색
   - "전체" 선택하여 검색
2. 날짜 범위 필터:
   - 시작일/종료일 설정 후 검색
   - 날짜 없이 검색 (최근 30일)
3. 예상 결과:
   - 필터에 맞는 결과만 표시
   - 필터 조합이 정상 작동

#### 1.3 에러 케이스 테스트
1. 빈 검색어로 검색
2. 존재하지 않는 검색어 (예: "asdkfjasldkfj")
3. 네트워크 오프라인 상태에서 검색
4. 예상 결과:
   - 적절한 에러 메시지 표시
   - 재시도 옵션 제공
   - 앱이 크래시되지 않음

#### 1.4 캐싱 테스트
1. 동일한 검색어로 2회 검색
2. 30분 이내 재검색
3. 예상 결과:
   - 첫 번째: API 호출
   - 두 번째: 캐시된 결과 반환 (빠른 응답)

#### 1.5 로그인 기능 테스트
1. 로그인 없이 검색 가능 확인
2. 로그인 후 북마크 기능 테스트
3. 예상 결과:
   - 비로그인: 검색 가능, 북마크 버튼 비활성
   - 로그인: 북마크 추가/제거 가능

## 2. 마이페이지 반응형 및 기능 테스트

### 테스트 시나리오

#### 2.1 레이아웃 테스트
1. 다양한 화면 크기에서 확인:
   - 모바일: 375px, 414px
   - 태블릿: 768px, 1024px
   - 데스크톱: 1280px, 1920px
2. 예상 결과:
   - 모든 화면 크기에서 레이아웃 깨짐 없음
   - 여백과 타이포그래피가 일관성 있게 유지
   - 미니멀 클린 스타일 유지

#### 2.2 탭 전환 테스트
1. "신청한 모임" 탭 클릭
2. "내 게시글" 탭 클릭
3. "본인인증 정보" 탭 클릭
4. "회사 소개" 탭 클릭 (후원 회원만)
5. 예상 결과:
   - 탭 전환이 부드럽게 작동
   - 각 탭의 컨텐츠가 정상 표시
   - 활성 탭 스타일 적용

#### 2.3 프로필 편집 테스트
1. "개인정보 수정" 버튼 클릭
2. 각 필드 수정:
   - 이름, 회사명, 직책
   - 업종, 주소, 전화번호
   - 프로필 이미지 업로드
3. "저장" 버튼 클릭
4. "취소" 버튼 테스트
5. 예상 결과:
   - 모든 필드 수정 가능
   - 저장 시 변경사항 반영
   - 취소 시 원래 값으로 복원

#### 2.4 게시글 편집 테스트
1. "내 게시글" 탭에서 편집 버튼 클릭
2. 제목, 내용 수정
3. 이미지 추가/삭제 (카테고리별)
4. "수정" 버튼 클릭
5. 예상 결과:
   - 모달이 정상적으로 열림
   - 모든 필드 수정 가능
   - 이미지 업로드/삭제 정상 작동
   - 변경사항 저장 후 모달 닫힘

#### 2.5 본인인증 정보 테스트
1. "본인인증 정보" 탭 클릭
2. 인증된 정보 확인:
   - 이름, 전화번호
   - 생년월일, 성별
   - 인증 거래번호
3. 예상 결과:
   - 모든 정보가 명확하게 표시
   - 미니멀한 디자인 유지
   - 정보 구분이 명확

#### 2.6 회사 소개 테스트 (후원 회원)
1. "회사 소개" 탭 클릭
2. 대표 이미지 URL 입력
3. 회사 소개 텍스트 작성
4. 추가 사진 URL 입력 (최대 3장)
5. "저장하기" 버튼 클릭
6. 예상 결과:
   - 모든 필드 입력 가능
   - 이미지 미리보기 표시
   - 저장 후 데이터 유지

## 3. 통합 테스트

### 3.1 전체 플로우
1. 메인 페이지 → 입찰공고 → 검색 → 결과 확인
2. 로그인 → 마이페이지 → 프로필 수정
3. 커뮤니티 → 게시글 작성 → 마이페이지에서 확인 → 편집
4. 프로그램 신청 → 마이페이지에서 확인 → 취소

### 3.2 성능 테스트
1. Lighthouse 점수 확인:
   - Performance: 80+
   - Accessibility: 90+
   - Best Practices: 90+
   - SEO: 90+
2. 로딩 시간:
   - 초기 로딩: 3초 이내
   - API 응답: 2초 이내 (캐시 미스)
   - API 응답: 500ms 이내 (캐시 히트)
