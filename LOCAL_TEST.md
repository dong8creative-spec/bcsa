# 로컬에서 프로그램·기능 테스트하기

본인 터미널에서 개발 서버를 띄우고, 프로그램 신청 등 기능을 직접 확인하는 방법입니다.

---

## 1. 터미널에서 개발 서버 실행

프로젝트 루트에서 아래를 실행하세요.

```bash
cd /Users/donghnc/Documents/bcsa
npm run dev
```

정상이면 예시처럼 나옵니다.

```
  VITE v5.x.x  ready in xxx ms
  ➜  Local:   http://127.0.0.1:3000/
  ➜  press h + enter to show help
```

서버를 종료하려면 터미널에서 **Ctrl + C** 를 누르세요.

---

## 2. 브라우저에서 접속

1. 브라우저를 열고 주소창에 **http://127.0.0.1:3000** 입력 후 접속  
   (Vite가 자동으로 브라우저를 띄웠다면 그 창을 사용해도 됩니다.)
2. 메뉴에서 **「프로그램」**, **「커뮤니티」** 등 원하는 메뉴로 이동합니다.

---

## 3. 테스트 체크리스트

### 3.1 프로그램 — 신청 (무료 / 유료)

- [ ] **프로그램** 메뉴에서 프로그램 카드의 **신청** 클릭
- [ ] 모달에서 **신청사유**, **사전질문 1**, **사전질문 2** 입력
- [ ] **무료 프로그램**: 버튼이 **「신청하기」** 로 보이고, 클릭 시 신청 완료
- [ ] **유료 프로그램**(Admin에서 신청 비용 입력한 경우): **「신청 비용: N원」** 이 보이고, 버튼이 **「결제하기」** 로 보이는지 확인

### 3.2 마이페이지

- [ ] 로그인 후 **마이페이지**에서 신청한 모임·내 게시글·본인인증 정보 확인

---

## 4. 포트가 이미 사용 중일 때

다른 프로그램이 3000번 포트를 쓰고 있으면 다음처럼 바꿀 수 있습니다.

```bash
# 예: 5173번 포트로 실행
npx vite --port 5173
```

이후 브라우저에서는 **http://127.0.0.1:5173** 으로 접속하면 됩니다.

---
