# 삭제 예정 파일 보관 폴더

이 폴더에는 사이트 운영에 직접적으로 필요하지 않은 파일들이 보관되어 있습니다.

## 보관된 파일 목록

### 가이드 문서들
- `DEPLOYMENT_GUIDE.md` - 배포 가이드
- `FIREBASE_DEPLOY.md` - Firebase 배포 가이드
- `FIREBASE_ENV_SETUP.md` - Firebase 환경 변수 설정 가이드
- `FIREBASE_FUNCTIONS_SETUP.md` - Firebase Functions 설정 가이드
- `FIRESTORE_INDEXES_GUIDE.md` - Firestore 인덱스 생성 가이드
- `GITHUB_PAGES_DEPLOY.md` - GitHub Pages 배포 가이드
- `HOSTING_INFO_REQUIRED.md` - 호스팅케이알 서버 정보 요청 문서
- `HOSTING_PROXY_SETUP.md` - PHP 프록시 설정 가이드
- `PHP_PROXY_IMPLEMENTATION_SUMMARY.md` - PHP 프록시 구현 요약
- `QUICK_FIREBASE_SETUP.md` - Firebase 빠른 설정 가이드
- `QUICK_START.md` - 빠른 시작 가이드
- `README_업로드가이드.md` - 업로드 가이드
- `REMOVE_TEST_ACCOUNTS.md` - 테스트 계정 제거 가이드
- `TESTING_GUIDE.md` - 테스트 가이드

### 예제 파일들
- `config.example.js` - 설정 파일 예제
- `firebase-config.example.js` - Firebase 설정 파일 예제

### 개발 서버 파일들
- `server.js` - Express 개발 서버 (로컬 개발용)
- `http-server.js` - HTTP 정적 파일 서버 (로컬 개발용)

## 참고

이 파일들은 나중에 참고가 필요할 수 있으므로 삭제하지 않고 보관합니다.
필요한 경우 이 폴더에서 파일을 다시 가져올 수 있습니다.

## 완전히 삭제하려면

이 폴더 전체를 삭제해도 사이트 운영에는 영향이 없습니다.
다만 나중에 참고가 필요할 수 있으므로 보관을 권장합니다.
